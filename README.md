## 🚀 Getting Started

### View Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/YosephEndale/YosephEndale.github.io.git
   cd YosephEndale.github.io
   ```

2. Open in your browser:
   ```bash
   # Using Python
   python3 -m http.server 8000
   
   # Or using Node.js http-server
   npx http-server
   ```

3. Navigate to `http://localhost:8000`
